# 🎉 新功能快速指南

## 主要更新

### 1. ✨ 自動預設標註框
- **之前**：需要點擊「開始標註」→ 點擊圖片添加點 → 點擊「完成標註」
- **現在**：上傳圖片後**自動創建矩形標註框**，立即可用！

### 2. ➕ 點擊邊線添加節點
- **操作**：在藍色邊線上點擊任意位置
- **效果**：自動在該位置插入新的控制節點
- **用途**：創建複雜的不規則多邊形

### 3. 🎯 簡化的界面
- **移除**：「開始標註」和「完成標註」按鈕
- **新增**：「重置標註」按鈕
- **結果**：更直觀，更易用

### 4. 🗑️ 刪除圖片功能 (NEW!)
- **位置**：圖片名稱右側的「刪除」按鈕
- **操作**：點擊即可刪除該圖片
- **效果**：立即從列表中移除圖片

### 5. 📐 固定圖片尺寸 (UPDATED!)
- **原則**：固定寬度 600px
- **高度**：根據比例自動計算（不設限）
- **用途**：統一圖片顯示尺寸

## 快速開始

### 步驟 1：上傳圖片
```
點擊「上傳圖片」→ 選擇圖片
```
✅ 自動創建矩形標註框

### 步驟 2：調整標註
**選項 A - 拖動節點**
```
點擊節點 → 拖動到新位置
```

**選項 B - 添加新節點**
```
將鼠標移到邊線上 → 光標變為十字 → 點擊
```

**選項 C - 重置**
```
點擊「重置標註」按鈕 → 恢復預設矩形
```

### 步驟 3：保存
```
點擊「保存標註」→ 數據輸出到控制台
```

## 鼠標光標變化

| 位置 | 光標 | 操作 |
|------|------|------|
| 控制節點上 | ✋ move | 可拖動 |
| 邊線上 | ➕ crosshair | 可添加節點 |
| 其他區域 | ➡️ default | 無操作 |

## 示例：從矩形到五邊形

```
1. 上傳圖片
   → 自動創建矩形（4個節點）

2. 在上邊線中點點擊
   → 添加第5個節點

3. 拖動新節點向上
   → 形成五邊形！
```

## 圖片管理

### 刪除圖片
```
方式 1: 點擊圖片名稱右側的「刪除」按鈕
方式 2: 在上傳列表中移除文件
```

### 圖片顯示
- **寬度**：固定 600px（或原始寬度，取較小值）
- **高度**：根據圖片比例自動計算
- **布局**：網格布局，最多3列
- **統一性**：所有圖片寬度一致，便於比較
